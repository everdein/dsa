# Trees

## What Is a Tree?

A tree is a hierarchical data structure made of nodes connected by edges.

## Why Trees Matter

Trees model parent-child relationships and are used in file systems, DOM structures, and decision-making problems.

## Common Operations

- DFS traversal
- BFS traversal
- Insert/delete nodes
- Find min/max values

## Example

```javascript
class TreeNode {
  constructor(value) {
    this.value = value;
    this.left = null;
    this.right = null;
  }
}
```
