# Graphs

## What Is a Graph?

A graph is a collection of nodes connected by edges. Graphs model networks, relationships, and state transitions.

## Why Graphs Matter

Graphs appear in routing, dependency mapping, social networks, and search problems.

## Common Operations

- DFS
- BFS
- Shortest path
- Cycle detection

## Example

```javascript
const graph = {
  A: ['B', 'C'],
  B: ['D'],
  C: ['D']
};
```
