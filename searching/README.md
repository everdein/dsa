# Searching

## What Is Searching?

Searching is the process of finding an item in a collection.

## Common Techniques

- Linear search
- Binary search
- Depth-first and breadth-first search for graphs and trees

## Example

```javascript
function linearSearch(arr, target) {
  return arr.indexOf(target);
}
```

## Practice Exercises

- Search in an unsorted array
- Search in a sorted array with binary search
