# Sorting

## What Is Sorting?

Sorting arranges data into a defined order such as ascending or descending.

## Why Sorting Matters

Many algorithms depend on sorted inputs, and sorting is a common interview topic.

## Common Techniques

- Bubble sort
- Selection sort
- Insertion sort
- Merge sort
- Quick sort

## Example

```javascript
const sorted = [3, 1, 2].sort((a, b) => a - b);
```

## Practice Exercises

- Sort an array
- Sort by custom criteria
- Merge two sorted arrays
