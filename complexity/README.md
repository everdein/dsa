# Complexity Analysis

## What Is Complexity Analysis?

Complexity analysis measures how an algorithm scales as the amount of input grows.

## Why It Matters

It helps compare algorithms and choose solutions that will perform well on large inputs.

## Common Notations

- O(1): constant time
- O(log n): logarithmic time
- O(n): linear time
- O(n log n): linearithmic time
- O(n^2): quadratic time

## Practice Focus

- Time complexity
- Space complexity
- Best, average, and worst case
