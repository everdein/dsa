# Matrices

## What Is a Matrix?

A matrix is a two-dimensional array of values arranged in rows and columns.

## Why Matrices Matter

Matrices are useful in grid-based problems, image processing, dynamic programming, and graph representations.

## Common Operations

- Traverse rows and columns
- Rotate the matrix
- Sum rows or columns
- Find paths in a grid

## Example

```javascript
const matrix = [[1, 2], [3, 4]];
console.log(matrix[1][0]);
```
